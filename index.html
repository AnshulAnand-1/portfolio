Project 4

Low Light Image Enhancement System

Developed a deep learning-based image enhancement system to improve visibility in low-light images using Retinex-inspired Transformer architecture. The model enhances brightness, reduces noise, and preserves natural colors and fine details. Trained and evaluated on LOL and SID datasets, achieving 26.23 dB PSNR and 0.89 SSIM, outperforming several existing methods. Built using Python, PyTorch, OpenCV, and Computer Vision techniques for efficient and high-quality image restoration.
